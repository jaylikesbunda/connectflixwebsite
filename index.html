Features

Grid & List View

Toggle between grid and list views to explore your entire media collection at a glance.

Detailed Media View

Click on media covers to access detailed information, including cast, synopsis, and more.

Data Integration

Automatically fetch and update data from The Movie Database, IMDb, and Rotten Tomatoes.

Price Tracking

Track average prices with eBay integration and observe historical trends for your collection.

Lending Management

Set media as lent, add lendee’s name, due date, and get reminders to ask for it back.

Barcode Scanning

Scan barcodes using smartphones, webcams, or USB scanners to easily add media to your collection.

Modern UI

Enjoy a sleek, modern browser UI inspired by Plex, with light/dark themes and easy navigation.

Server-Based Access

Run CollectFlix as a server on your PC, accessible through a web UI on your network.

Automatic Updates

Keep CollectFlix up to date with in-app update notifications and seamless software updates.

How It Works

CollectFlix makes managing your media collection simple and effective:

  • Grid view displays your entire collection, with adjustable display modes to toggle between grid and list views.
  • Click on a media cover to access a detailed view, pulling information from The Movie Database, IMDb, and Rotten Tomatoes.
  • Automatic data fetching keeps your media information up to date, with integration with eBay to track the average price of your media.
  • Price logging allows you to observe historical price trends for your collection.
  • Manage lending by setting media items as lent, including details such as the lendee’s name and due date, with automated reminders to ask for it back.
  • Scan barcodes using your smartphone, webcam, or USB barcode scanner, and manually enter or correct information if needed.
  • CollectFlix runs as a server on your Windows PC, allowing local network devices to access the database and media scanning.
  • The user interface is designed to be clean, modern, and accessible, with a theme switcher for light and dark modes.

About CollectFlix

CollectFlix is an open-source application designed to help you manage your movie and media collection with ease. Whether you're a casual collector or a dedicated enthusiast, CollectFlix provides all the tools you need to organize, track, and value your collection.
